# AION Project.com

![](https://i.imgur.com/fJ98rh9.png)

The official website for the AION Project. It surfaces the project's game data and assets in the browser: a searchable database, player and game information, and a 3D engine page for exploring models and maps. It's also reachable from inside the game client.

## Features

- Searchable database of game content: items, npcs, quests, and more.
- Item stats, lore, and broker listings.
- Character profiles and player information.
- Server and game information in one place.
- Engine page that loads files reversed from the game's CGF and LST formats via [IzFF](https://github.com/Iswenzz/IzFF).
- Explore the game's models and maps directly in the browser.

## Building

_Pre-Requisites:_

1. [Node](https://nodejs.org/en/)
2. [Yarn](https://yarnpkg.com/)
